# Nillion Builder Bounty Submission: WebAuthn + Ed25519 PoC
## Overview
This project provides a fully functional Proof of Concept (PoC) for an authentication system using WebAuthn to verify Ed25519 key pairs.
## Technical Highlights
- **Algorithm:** COSE Algorithm -8 (EdDSA with Ed25519).
- **Secure Architecture:** Native browser WebAuthn API.
## How to Test
1. Run `python server.py`.
2. Visit `http://localhost:8000`.
3. Test the registration/authentication buttons.
